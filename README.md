# Silvia Föger — Künstlerinnen-Homepage

Statische Single-Page-Site für die Tiroler Künstlerin Silvia Föger, um handgefertigte Keramik, Poesie und Fotografie zu präsentieren und Besucher*innen zu ihrem Shopify-Shop zu führen.

## Struktur

- `index.html` — semantisches Layout mit Hero, Geschichte, Schwerpunktbereichen, Galerie, Shop-CTA, Newsletter und Footer.
- `assets/css/style.css` — individuelle, von Tonfarben inspirierte Gestaltung samt responsivem Grid und mobiler Navigation.
- `assets/js/main.js` — Steuerung des mobilen Menüs und dynamisches Copyright.

## Schnellstart

Das Projekt basiert ausschließlich auf HTML/CSS/JS. Starte eine lokale Vorschau mit einem beliebigen Static-Server, zum Beispiel:

```bash
npx serve .
```

Anschließend im Browser `http://localhost:3000` (oder die im Terminal angegebene URL) aufrufen.

## Anpassungstipps

- Ersetze die Unsplash-Bild-URLs in `index.html` durch Silvias eigene Fotografien oder Shopify-Assets.
- Aktualisiere bei Bedarf Kontaktangaben, Social-Links und das Newsletter-Ziel.
- Um Shopify-Produkte direkt einzubinden, kann der Shop-CTA durch ein Shopify-Buy-Button-Snippet ersetzt werden.
